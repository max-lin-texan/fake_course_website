課程資訊

本課介紹平行計算：平行架構、平行演算法與其效率分析。實作以 C 為主，亦涵蓋 OpenMP、MPI、CUDA（選配 NCCL）。
課程將由理論（平行複雜度、效率）逐步轉向實務（排序、圖論、矩陣演算法等在 CUDA 架構上的應用）。
